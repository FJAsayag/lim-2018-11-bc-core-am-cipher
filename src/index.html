CÓDIGO SECRETO

Un poco de historia...

En los tiempos de Julio César, se usaba este cifrado para comunicarse de manera secreta...

Ahora podrías hacerlo tú.


Instrucciones

  1. Ingresa el mensaje que deseas cifrar, o sea, transformar para hacerlo secreto.
  2. Ingresa el número clave de tu gusto para la creación de tu código secreto. Recuerda que este número clave es esencial para crear y entender el texto secreto.
  3. Dale click al botón para ver cómo se ve lo que pusiste en el cuadro de texto



Lo que podrías hacer...

  • Envía mensajes secretos a tus amigxs con ayuda de Código Secreto. ¡Es más divertido!
  • Mandar mensajes de amor secretos que solo esa personita que te gusta entenderá.
  • ¿Has pensado en comunicarte con tu papá o mamá de manera más graciosa? Rétalos con Código Secreto.
